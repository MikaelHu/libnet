This is a net dev package, based on asio.
This libNet includes Tcp server/client, Udp server/client, Http server/client, multicast etc.
These servers and clients all support high concurrency, with low latency, and can be used to MSG transfer,
 files transfer and other usage scenario.
The connection supports openssl, and the data stream can be verified with MD5.
